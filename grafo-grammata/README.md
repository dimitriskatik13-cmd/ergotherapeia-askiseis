# ΣΥΝΟΙΔΑ · «Γράφω Γράμματα»

Κλινικό εργαλείο (PWA για iPad) όπου το παιδί εξασκεί τη γραφή των ελληνικών
γραμμάτων με το δάχτυλο ή το Apple Pencil. Υπηρετεί ταυτόχρονα **εργοθεραπευτικό**
στόχο (σωστή φορά & σειρά γραμμών ως αυτοματοποιημένο κινητικό μοτίβο) και
**λογοθεραπευτικό** στόχο (στην ολοκλήρωση κάθε γράμματος ακούγεται το **φώνημα**).

> Αυτή η εφαρμογή ζει στον υποφάκελο `grafo-grammata/` του repo. Είναι ξεχωριστή
> από την εφαρμογή «Εργοθεραπευτικές Ασκήσεις» που βρίσκεται στη ρίζα.

## Πρόσβαση / Εγκατάσταση στο iPad

1. Deploy μέσω **GitHub Pages** (ήδη ενεργό για το repo).
2. Άνοιξε στο iPad: `https://<user>.github.io/ergotherapeia-askiseis/grafo-grammata/`
3. Safari → **Κοινή χρήση → Προσθήκη στην Αρχική οθόνη** → εικονίδιο **ΣΥΝΟΙΔΑ**.
4. Ανοίγει fullscreen και δουλεύει **offline** μετά την πρώτη φόρτωση.

## Τρόποι λειτουργίας

| Mode | Κουμπί | Τι κάνει |
|---|---|---|
| Επίδειξη | **Δείξε μου** | Το γράμμα «γράφεται» μόνο του με animated «μολύβι», σωστή φορά & σειρά. |
| Ιχνηλάτηση | **Ακολούθησε** | Το παιδί γράφει πάνω σε αχνό οδηγό με αριθμούς ① & βέλη. |
| Σταδιακή απόσυρση | **Σταδιακή** | 4 επίπεδα βοήθειας (fading) — errorless learning → ανεξαρτησία. |
| Ελεύθερη | **Ελεύθερη** | Χωρίς οδηγό· ολοκλήρωση με το κουμπί ✓ του θεραπευτή. |

**Επίπεδα βοήθειας (Σταδιακή):** 1 = οδηγός+αριθμοί+βέλη · 2 = −βέλη · 3 = μόνο
σημεία έναρξης · 4 = χωρίς οδηγό.

**Ολοκλήρωση → φώνημα:** σε ιχνηλάτηση/σταδιακή ανιχνεύεται αυτόματα όταν καλυφθούν
σωστά όλα τα strokes· σε **όλα** τα modes το κουμπί **✓ Ολοκλήρωση** του θεραπευτή
ενεργοποιεί το φώνημα. Επιβράβευση διακριτική (απαλό glow + 1–2 μπαλόνια), όχι
θορυβώδη εφέ/πόντοι.

## Ρυθμίσεις θεραπευτή (⚙️)

Στοχευμένα γράμματα (υποσύνολο για τη συνεδρία) · Πεζά/Κεφαλαία · επίπεδο βοήθειας ·
**αυστηρότητα ελέγχου** · διαβάθμιση (**πάχος**, **πίεση** Pencil, **μέγεθος**,
**γραμμές** τετραδίου, **ταχύτητα** επίδειξης) · **δεξιόχειρας/αριστερόχειρας**.

## Ήχος — αποσυνδεδεμένος (decoupled)

Η εφαρμογή **δεν** κάνει live TTS. Παίζει προ-παραγμένα τοπικά αρχεία από τον φάκελο
`sounds/` με **σταθερή ονοματολογία ανά φώνημα** (`a.mp3`, `s.mp3`, `m.mp3`, …). Τα
ομόηχα γράμματα (η/ι/υ → `i`, ο/ω → `o`) και τα ζεύγη πεζό/κεφαλαίο μοιράζονται αρχείο.

**Αναβάθμιση σε φυσική φωνή χωρίς αλλαγή κώδικα:** απλώς αντικατέστησε τα αρχεία στο
`sounds/` με ίδια ονόματα (π.χ. cloud TTS με SSML phoneme tags). Η ονοματολογία **δεν
αλλάζει ποτέ**.

Τρέχουσα μέθοδος v1: **eSpeak-NG** (offline, χωρίς κόστος/API). Ρομποτικός αλλά
φωνημικά σωστός ήχος.

## Ομαλότητα γραφής (engine)

Pointer Events · `getCoalescedEvents()` (πυκνά σημεία Pencil) · εξομάλυνση quadratic
Bézier μέσω midpoints · `devicePixelRatio` scaling · `touch-action:none` +
`preventDefault()` + `setPointerCapture()` (η σελίδα δεν scroll-άρει/zoom-άρει) ·
προαιρετικό μεταβλητό πάχος με `pressure`.

## Απόρρητο

Καμία αποθήκευση/αποστολή δεδομένων παιδιού. Όλα τρέχουν τοπικά. Στο localStorage
μένουν **μόνο** οι ρυθμίσεις θεραπευτή.

## Αναπαραγωγή assets (reproducible)

```bash
# Φωνήματα (απαιτεί espeak-ng + ffmpeg):
bash tools/gen_phonemes.sh
# Placeholder logo/icon (απαιτεί python3 + Pillow):
python3 tools/gen_logo.py
# Τοπική προεπισκόπηση:
python3 -m http.server 8000   # → http://localhost:8000/grafo-grammata/
```
